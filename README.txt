Projet Sudoku 2024-2025 softwares engineering

Fonctionnement :
    1.Récupérer le fichier projetsudoku.jar dans ./out/artifacts/projetsudoku_jar/
    2.Executer le fichier avec : java -jar projetsudoku.jar
    3.Entrer la règle que vous voulez appliquer (entre 1 et 3)
    4.Entrer le chemin vers la grille à résoudre
    5.Si la grille n'est pas résolue du premier coup, le programme demandera une case a remplir correctement pour continuer.

Le format du fichier grille :
Le fichier doit être un fichier txt avec, sur chaque ligne les 9 valeurs
des colonnes toutes séparées par des virgules , 0 représente le vide. (Si besoin , regarder les exemples de grille dans le dossier)

Exemple d'utilisation :

    java -jar projetsudoku.jar

    Quelle règle appliquer ? (entre 1 et 3) :
    1

    Chemin vers la grille à résoudre :
    grilles/grille1.txt
